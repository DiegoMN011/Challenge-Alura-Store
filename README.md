# Challenge-Alura-Store

# 📊 Análise de Dados - Alura Store

## 📌 Descrição do Projeto

Este projeto consiste em uma análise exploratória de dados de quatro lojas fictícias da **Alura Store**.
O objetivo da análise é avaliar o desempenho das lojas com base em diferentes métricas e, ao final, recomendar **qual loja o Sr. João deve vender**.

A análise foi realizada utilizando **Python**, com as bibliotecas **Pandas** para manipulação de dados e **Matplotlib** para visualização.

---

## 🎯 Objetivo da Análise

Comparar o desempenho das quatro lojas considerando:

* Faturamento total
* Categorias de produtos mais vendidas
* Média de avaliação dos clientes
* Produtos mais e menos vendidos
* Frete médio por loja

Com base nessas informações, foi possível identificar **qual loja apresenta o melhor desempenho geral**.

---

## 🛠️ Tecnologias Utilizadas

* Python
* Pandas
* Matplotlib
* Google Colab / Jupyter Notebook

---

## 📂 Estrutura da Análise

A análise foi dividida nas seguintes etapas:

### 1️⃣ Importação e carregamento dos dados

Os dados foram carregados diretamente de arquivos CSV hospedados no GitHub utilizando a biblioteca **Pandas**.

### 2️⃣ Análise do faturamento

Foi calculado o faturamento total de cada loja através da soma da coluna **Preço**.

### 3️⃣ Vendas por categoria

Foram analisadas as categorias de produtos mais vendidas em cada loja utilizando contagem de frequência.

### 4️⃣ Média de avaliação das lojas

Foi calculada a média das avaliações feitas pelos clientes para cada loja.

### 5️⃣ Produtos mais e menos vendidos

Foi identificado o produto com maior número de vendas e o produto com menor número de vendas em cada loja.

### 6️⃣ Frete médio por loja

Foi calculado o valor médio de frete cobrado em cada loja.

---

## 📊 Visualização de Dados

Foram criados gráficos utilizando **Matplotlib** para facilitar a interpretação dos resultados:

* Gráfico de **faturamento por loja**
* Gráfico de **avaliação média das lojas**
* Gráfico de **distribuição de categorias de produtos**

Essas visualizações ajudam a identificar padrões e diferenças entre as lojas.

---

## 📈 Conclusão

Após a análise dos dados, foi possível comparar o desempenho das quatro lojas considerando faturamento, satisfação dos clientes e custos de frete.

Com base nessas informações, foi feita uma recomendação sobre **qual loja o Sr. João deve vender**, priorizando o melhor desempenho geral entre os indicadores analisados.
